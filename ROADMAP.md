# Project Roadmap

This roadmap outlines the milestones and goals planned for WexSay. As we transition from our temporary Public Beta stage to our official production releases, these targets help guide development.

---

## 📍 Current Phase: Public Beta Program (Q2 - Q3 2026)
* **Goal:** Verify messaging engine stability and calling reliability across diverse Android hardware configurations.
* [x] **Milestone 1:** Launch initial beta build (`com.expde.chitchat`).
* [x] **Milestone 2:** Hardening local security (EncryptedSharedPreferences) and patching sync errors.
* [ ] **Milestone 3:** Collect community feedback, resolve audio routing delays on WebRTC, and tune push notifications.
* [ ] **Milestone 4:** Optimize asset caching and lower background battery consumption.

---

## 🚀 Phase 2: Play Store Production Release (Q4 2026)
* **Goal:** Re-package the application and launch on the Google Play Store under our stable production profile.
* [ ] **Milestone 1:** Verify the data migration mechanism (logging into the new stable application imports previous backend profile settings).
* [ ] **Milestone 2:** Create the official release package profile (`com.wexde.wexsay`).
* [ ] **Milestone 3:** Set up legal pages, Google Play Console compliance, and privacy policies.
* [ ] **Milestone 4:** Initial public deployment on the Google Play Store.

---

## 🌟 Phase 3: Platform Expansion (2027)
* **Goal:** Extend the WexSay messaging ecosystem to desktop, web, and other platforms.
* [ ] **Milestone 1:** Development of secure Desktop clients (Windows/macOS/Linux).
* [ ] **Milestone 2:** Development of a responsive Web client.
* [ ] **Milestone 3:** Localization support for multiple languages (Spanish, German, Bengali, Hindi, Arabic, etc.).
* [ ] **Milestone 4:** Enterprise-oriented configurations (Custom group controls, moderator audits, file attachment size upgrades).

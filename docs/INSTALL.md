# Installation Guide

This document provides detailed instructions for installing the WexSay Android application during our Public Beta phase.

---

## 📱 Installation Methods

### Method 1: Direct APK Download (Recommended)
This is the simplest way to install WexSay on your Android device.

1. **Download:** Open your device's browser, navigate to the [Releases](https://github.com/Wex-DE/WexSay/releases) section of this repository, and download the latest `.apk` file.
2. **Open:** Once downloaded, tap the notification or navigate to your device's **Downloads** folder and tap the `wexsay-...-release.apk` file.
3. **Allow Installation:** If this is your first time installing an independent APK, Android will ask for confirmation.
   * Tap **Settings** on the prompt.
   * Toggle **"Allow from this source"** for your browser or file manager.
   * Tap **Back** to return to the installer.
4. **Complete Setup:** Tap **Install**. Once installation finishes, tap **Open** to launch WexSay!

---

## 🛠️ Resolving Play Protect Warnings

Because WexSay's Public Beta is not yet published directly through the Google Play Store registry, Android's Play Protect service may display a warning:
* **Message:** *"Blocked by Play Protect — Unknown developer."*
* **How to bypass:** Tap **"More details"** on the popup, then tap **"Install anyway"**.

---

## 🔄 Updating the Application

* **Manual Updates Required:** The beta application uses a temporary package name (`com.expde.chitchat`). It **will not** receive automatic updates from the Google Play Store.
* **How to Update:** When we release a new beta build, download the new APK from the GitHub releases page and install it directly over the existing app. Your accounts and chat histories will be preserved.

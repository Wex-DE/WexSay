# Open Source Acknowledgements

The WexSay Android application relies on several open-source libraries and frameworks. We are grateful to the authors and maintainers of these projects for their contributions to the developer ecosystem.

---

## 📱 Key Open Source Libraries Used

### Android Foundation & UI
* **Jetpack Compose / AndroidX Suite:** Licensed under the [Apache License 2.0](https://www.apache.org/licenses/LICENSE-2.0). Provides our modern declarative UI design architecture.
* **Coil:** Licensed under the [Apache License 2.0](https://github.com/coil-kt/coil/blob/main/LICENSE.txt). Enforces fast image and animated WebP loading.
* **AXEmojiView:** Licensed under the [Apache License 2.0](https://github.com/Aghajari/AXEmojiView). Powers our advanced native custom emoji selectors.

### Networking & Database
* **Retrofit:** Licensed under the [Apache License 2.0](https://github.com/square/retrofit/blob/master/LICENSE.txt). Powers RESTful API HTTP calls.
* **OkHttp:** Licensed under the [Apache License 2.0](https://github.com/square/okhttp/blob/master/LICENSE.txt). Handles underlying connection pooling.
* **Socket.io Client Java:** Licensed under the [MIT License](https://github.com/socketio/socket.io-client-java/blob/master/LICENSE). Powers persistent WebSocket chats.
* **Room Database:** Licensed under the [Apache License 2.0](https://www.apache.org/licenses/LICENSE-2.0). Provides local SQLite database caching.

### Communications & Media
* **Conversations WebRTC Android:** Licensed under the [Apache License 2.0](https://github.com/iNPUTmice/webrtc-android). Provides standard audio and video calling engine integrations.
* **ZXing Android Embedded:** Licensed under the [Apache License 2.0](https://github.com/journeyapps/zxing-android-embedded). Handles QR Code scanning for instant client connections.
